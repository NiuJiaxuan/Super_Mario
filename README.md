## Technologies Used
- **Language:** C#
- **Framework:** MonoGame


## Installation & Setup
Clone this repository:
   ```bash
   git clone https://github.com/NiuJiaxuan/Super_Mario.git
   ```

## How to Play 

- **Arrow keys:** Move Mario
- **Space:** Shoot Fireball
- **M**: Mute
- **R**: Reset
- **P**: Pause
